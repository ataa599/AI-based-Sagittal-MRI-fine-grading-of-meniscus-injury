AI-Based Sagittal MRI Fine-Grading of Meniscus Injury

About This Project

This application uses advanced deep learning techniques to analyze sagittal MRI images and provide fine-grained classification of meniscus injuries. The system processes DICOM images and identifies three key regions of the meniscus:

  • Posterior Horn: The back portion of the meniscus
  • Body: The central portion of the meniscus
  • Anterior Horn: The front portion of the meniscus

For each region, the AI model predicts the severity of the injury and provides a confidence score to assist medical professionals in diagnosis.

This option allows you to retrain the deep learning model from scratch using the complete pipeline. The training process includes:

  • Data Preprocessing: Creating new dataset from raw DICOM files and cropping regions of interest
  • Data Augmentation: Applying image transformations to increase dataset size and model robustness
  • Dataset Splitting: Dividing data into training and testing sets
  • Model Training: Training the neural network with validation and performance monitoring

Note: This process is time-intensive and requires significant computational resources. It will overwrite the existing trained model.
